# LitTrack - Personal Book Shelf Manager 📚

LitTrack is a secure, web-based application designed to help users track their reading progress. This project marks the completion of my deep dive into Core PHP and MySQL.

### ✨ Key Features
- **User Authentication:** Secure login and registration with password hashing (Bcrypt).
- **Personalized Dashboard:** Each user has their own private bookshelf.
- **Dynamic Search & Filter:** Quickly find books by title, author, or status (Reading, Completed, etc.).
- **Statistics:** Real-time summary of total, reading, and completed books.
- **Responsive UI:** Built with Bootstrap 5 for a seamless mobile/desktop experience.

### 🛠️ Tech Stack
- **Backend:** PHP (Procedural with Prepared Statements)
- **Database:** MySQL
- **Frontend:** Bootstrap 5, HTML5, CSS3

### 🚀 Lessons Learned
This project allowed me to master **Session Management**, **SQL Injection Prevention**, and building a **Relational Database** where data is linked to specific users.
